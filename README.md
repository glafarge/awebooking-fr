# Traduction française du plugin AweBooking

La structure de ce dépôt correspond à celle du dossier wp-content/

N'hésitez pas à faire des pull request pour améliorer les traductions.

Il existe plusieurs fichiers de traduction car le plugin a été conçu avec des dépendances (voir dossier vendor/).

Versions :

- AweBooking v3.0.0
- Wordpress v4.8


NB : les traduction ont été effectuées pour la plupart en utilisant (Google Translate)[https://translate.google.com/toolkit/workbench].
L'automatisme des traductions ne les rend pas précises, pour autant cela fait une bonne base à corriger.
